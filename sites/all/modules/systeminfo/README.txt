MODULE
------
System information


DESCRIPTION/FEATURES
--------------------
Displays information about the Drupal installation and system environment.


REQUIREMENTS
------------
Drupal 6.x


INSTALL/CONFIG
--------------
1. Move this folder into your modules directory.
2. Enable it from Administer >> Site building >> Modules >> Administration.
3. It's available now at Administer >> Reports >> System information.
4. Configure the display at Administer >> Site configuration >> System information.
